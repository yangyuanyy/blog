写自己的JavaScript工具－自定义事件

要点

  1. 传统事件
  2. 自定义事件

1. 传统事件

load、click、resize、readystatechange等等，都是 javaScript 客户端常见的事件。Window 对象、Docuement 对象和所有文档元素等都可以绑定事件处理程序。绑定方式可以是：

  1. 设置事件目标的属性，比如设置 window.onload、document.onclick、request.onreadystatechange 的值为事件处理程序；
  2. 设置 HTML 标签属性为事件处理程序；
  3. 通过 addEventListener 方法注册一个或多个事件处理程序。

通过事件目标属性和 HTML 标签注册的事件处理程序优先于通过 addEventListener 注册的事件处理程序，addEventListener 中注册的时间处理程序按照注册顺序调用。
