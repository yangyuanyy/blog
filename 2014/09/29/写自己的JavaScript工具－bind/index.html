bind 方法，顾名思义是将一个函数绑定到一个对象，同时会生成一个新的函数。执行这个新的函数，就像在被绑定的对象上调用这个方法一些样。

3. 实现完整的 bind 方法

然而，ECMAScript 5 中的 bind 方法不仅实现了上述功能，还实现了另外一个特性：
bind 除了接收一个对象为第一个参数以实现上述功能外，还可以接收更多额外参数，这些额外将被直接绑定到原函数的各个对应形参，可能还会剩余一些参数未被绑定，剩余的形参将由返回的新函数提供。

这种特性称为“柯里化”，即对于一个函数，如果固定某些参数，将得到一个接收余下参数的新函数。所以对于有两个变量的函数f( x, y ) = xy，如果固定了 y = 2，则得到有一个变量的新函数 g( x ) = 2x。
